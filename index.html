HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The <h1>, <h2>, <h3>, <h4>, <h5>, and <h6> elements represent the level of heading a given text block represents.
  • The <ul>, <ol>, and <li> elements represent unordered lists, ordered lists, and list items..
  • The <img> element contains information about images that are displayed on the webpage.

  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: stages the changes on the current branch, with the flag -A denoting all files
  • git commit -m "relevant comment": commits the changes, with the flag -m and associated comment keeping a record of the changes
  • git pull origin main: syncs local branch with the remote branch in Github
